# koda-lang


### Instalacion
El proyecto necesita python 3.7+
Actualmente no se tiene ninguna libreria de dependencia.

### Pruebas simples
Para ejecutar el proyecto yo recomiendo trabajar parte por parte. Es decir, ejecutar solo el lexer, parser y similares.

Un ejemplo seria
```bash
#ingresar al directorio src
cd ./src
```

```bash
#Y dentro de src ejecuto directamente el lexer
python3 ./lexer.py

```

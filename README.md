# koda-lang


### Instalacion

El gestor original de paquetes en pnpm en lugar de npm, para usar npm sin necesidad de instalar pnpm
puedes usar el siguiente comando:

``` bash
//instalara las dependencias
npm run i
```

Una vez instaladas las dependencias puedes empezar a trabajar usando
``` bash
npm run dev
```

Y podras ir viendo los cambios y errores en la terminal.
